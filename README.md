# 🎓 SeatMaster Pro

> **Seating, Reimagined**

AI 智能座位管家 — 让排座位，从此优雅。

---

## ✨ 核心特性

- 🎓 **教室平面图** — 50 座 4 大组布局（12 + 14 + 12 + 12）
- 🎰 **5 种点名模式** — 全班随机 / 按大组 / 按列行 / 均衡 / 连点
- 🪑 **AI 智能排座** — 身高 / 视力 / 性别规则
- 🕵️ **暗箱操作** — 表面随机，实际铁律
- 🌗 **深浅双主题** — 圆形扩散扫过动画
- 🎹 **关于界面** — 打字机效果展示团队成员
- 🔊 **程序化音效** — Web Audio API 生成
- 💾 **本地持久化** — localStorage 自动保存
- 📥 **导入导出** — JSON / CSV

## 🚀 快速开始

### 方式 1：浏览器直接打开（最简单）
双击 `index.html` 即可在浏览器中使用。

### 方式 2：使用启动器
双击 `启动.bat`，按提示选择启动方式。

### 方式 3：本地服务器（推荐，避免某些浏览器限制）
```bash
# 方式 A：使用 Python
python -m http.server 8000
# 然后访问 http://localhost:8000

# 方式 B：使用 Node.js
npx -y http-server -p 8000
```

### 方式 4：打包桌面应用
```bash
cd electron
npm install
npm run build:win    # 生成 .exe
```

## 📖 使用指南

### 1️⃣ 加载学生
- **演示数据**：点击"演示"按钮，一键加载 50 个示例学生
- **批量导入**：每行一个名字粘贴到文本框
- **JSON 导入**：使用 SaveManager 导出的数据
- **手动添加**：点击"添加学生"逐个编辑

### 2️⃣ 排座
- 进入"排座"页，开启规则（身高/视力/性别）后点击"一键智能排座"
- 如已设置暗箱预设，会优先满足预设约束

### 3️⃣ 点名
- 进入"点名"页，选择模式
- 点击"开始点名"触发老虎机动画
- 中签者会高亮在教室平面图上

### 4️⃣ 暗箱操作
- 进入"暗箱"页
- 新建方案 → 添加约束（强制分组/同桌/避让等）
- 排座时自动应用预设
- 学生看到的只是"随机"结果

### 5️⃣ 切换主题
- 点击右上角 🌙/☀️ 按钮
- 圆形从按钮位置扩散切换主题
